# Operational Scripts

This directory contains **reusable operational scripts** for Linux systems.

## Scope
- Automation for routine operations
- Diagnostics and inspection helpers
- Recovery and emergency scripts

## Principles
- Scripts must be **idempotent** when possible
- Safe to run multiple times
- Explicit output and clear failure modes

## Usage
- Review before execution
- Test in non-production environments first
- Adapt parameters to your environment

## Notes
- Favor readability over cleverness
- Avoid hidden side effects
- Include usage instructions in script headers
