# 7）周期性异常（固定时间点）

> 来源：`故障排查肌肉记忆.md`（旧文件）。

**用户感知**

```bash
watch -n 2 curl http://127.0.0.1:8080/
```

**故障注入**

```bash
echo '* * * * * yes > /dev/null' | crontab -
```

**运维起手（L1）**

```bash
date
uptime
crontab -l
```

**恢复**

```bash
crontab -r
pkill yes
```

---

# 训练执行纪律（不可违反）

- 必须从 **用户感知** 开始
    
- **禁止**提前敲 `systemctl` / `top`
    
- **禁止**跳过任一步
    
- 每个场景 **必须恢复**
    
- 只练顺序和手感，不问“为什么”
