# 5）偶发失败（一会儿好一会儿坏）

> 来源：`故障排查肌肉记忆.md`（旧文件）。

**用户感知**

```bash
for i in {1..5}; do curl http://127.0.0.1:8080/; done
```

**故障注入**

```bash
sudo systemctl stop nginx
sleep 3
sudo systemctl start nginx
```

**运维起手（L1）**

```bash
for i in {1..5}; do curl -i http://127.0.0.1:8080/; done
systemctl status nginx
top
```

**恢复**

```bash
sudo systemctl restart nginx
```
