# 4）超时 / 一直转圈

> 来源：`故障排查肌肉记忆.md`（旧文件）。

**用户感知**

```bash
curl --max-time 3 http://127.0.0.1:8080/
```

**故障注入**

```bash
sudo nft add rule inet filter input tcp dport 8080 drop
```

**运维起手（L1）**

```bash
curl --max-time 3 http://127.0.0.1:8080/
top
ss -lnt
```

**恢复**

```bash
sudo nft delete rule inet filter input tcp dport 8080 drop
```
